Our repo is the submission for the OOP project, VNU-UET. Educational purposes only.

Features: 

Classical style gameplay

Bricks and powerups in multiple types

Distinct InputHandler for Paddle

Level maps are JSON files, easy for extracting

Seperate renderer

Multithreading: Game logic, Game View, Sound

JUnit Test for core components
